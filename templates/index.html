Codes d'erreur HTTP

200

OK. La requête a réussi.

201

Créé. La requête a abouti et une nouvelle ressource a été créée.

202

Accepté. La requête a été acceptée pour traitement, mais n'est pas encore terminée.

204

Aucun contenu. La requête a réussi mais ne retourne aucun contenu.

300

Choix multiples. Plusieurs options sont disponibles pour la ressource demandée.

301

Déplacé définitivement. La ressource a une nouvelle URL permanente.

302

Déplacement temporaire. La ressource est temporairement à une autre URL.

304

Non modifié. La ressource demandée n’a pas été modifiée depuis la dernière requête.

307

Redirection temporaire. La ressource est temporairement redirigée à une autre URL.

400

Requête invalide. Le serveur n'a pas compris la requête.

401

Non autorisé. Une authentification est requise.

403

Interdit. Accès refusé à la ressource.

404

Non trouvé. La ressource demandée est introuvable.

405

Méthode non autorisée. La méthode utilisée n'est pas autorisée pour cette ressource.

408

Délai d'attente dépassé. Le serveur a expiré en attendant une requête.

429

Trop de requêtes. Le client a envoyé trop de requêtes en un temps limité.

500

Erreur interne du serveur. Une erreur inattendue s'est produite.

501

Non implémenté. Le serveur ne prend pas en charge la méthode demandée.

502

Passerelle incorrecte. Une réponse invalide a été reçue.

503

Service indisponible. Le serveur est temporairement surchargé ou en maintenance.

504

Délai de la passerelle dépassé. Le serveur n'a pas répondu à temps.

505

Version HTTP non prise en charge. Le serveur ne prend pas en charge la version HTTP utilisée.

Résumé SQLite3 avec Python et Flask

Commandes SQLite

.read <file.sql> - Charger un fichier SQL.

.tables - Lister toutes les tables.

.schema <table> - Afficher le schéma d'une table.

.exit - Quitter la console SQLite.

Exemples de requêtes SQL

CREATE TABLE users (id INTEGER PRIMARY KEY, name TEXT);

INSERT INTO users (name) VALUES ('Alice');

SELECT * FROM users;

UPDATE users SET name='Bob' WHERE id=1;

DELETE FROM users WHERE id=1;

Flask et SQLite

sqlite3.connect('database.db') - Se connecter à la base.

cursor.execute('SQL_QUERY') - Exécuter une requête SQL.

connection.commit() - Sauvegarder les modifications.

connection.close() - Fermer la connexion.
